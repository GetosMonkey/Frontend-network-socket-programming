## Run information

To run the project, navigate to the project root directory in your terminal and execute the following commands:

| Action | Command | Description |
| :--- | :--- | :--- |
| **Start Server** | `python server/tcp_server.py` | Launches the multi-threaded TCP server. |
| **Start Client** | `python client/tcp_client.py` | Launches the client interface to connect to the server. |
| **Terminate All** | `taskkill /F /IM python.exe` | Force-closes all active Python processes (Windows). |
